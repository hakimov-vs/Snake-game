# Snake-game

A classic 2D Snake game built with HTML5 Canvas and JavaScript.

## Description

This project is a recreation of the classic Snake game using web technologies. Navigate your snake through the game board, eat food to grow longer, and avoid hitting the walls or yourself. The game features smooth animations powered by HTML5 Canvas and interactive controls.

## Technologies

- **JavaScript** (68.5%) - Game logic and mechanics
- **HTML** (26.3%) - Game canvas and structure
- **CSS** (5.2%) - Styling and layout

## Features

- Smooth snake movement and animations
- Food collection mechanics
- Collision detection (walls and self)
- Score tracking
- Responsive controls

## How to Play

1. Use arrow keys to control the snake's direction
2. Eat the food to grow longer and increase your score
3. Avoid hitting the walls and the snake's own body
4. Try to achieve the highest score possible

## Installation

Simply open the game in your web browser to play. No additional dependencies or installation required.

## Getting Started

1. Clone this repository
2. Open the HTML file in your preferred web browser
3. Start playing!
